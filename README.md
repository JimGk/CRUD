# CRUD
Simple program Create Update Delete.
This program was uploaded after its completion, from the excersices of CodeGym.
The following arguments are acceptable witout any restrictions to the number of inputs.

-c name1 sex1 bd1 name2 sex2 bd2 ...        -----Create new person and add him to the list, the -c also prints the id of the new person at the end.
-u id1 name1 sex1 bd1 id2 name2 sex2 bd2 ...-----Update existing person
-d id1 id2 id3 id4 ...                      -----Setting values of people's selected id to null
-i id1 id2 id3 id4 ...                      -----Display the whole list
